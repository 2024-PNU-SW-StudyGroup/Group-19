## css 속성을 스스로 알아내기
뭔가 디자인을 바꾸고 싶은 부분이 어떤 태그인지 알아야함.

### 원하는 부분 크기 바꾸기
검색어: Css text size property

Font-size를 이용해 제목 크기 바꾸기
h1{ font-size:45px;  }

### 정렬하기
가운데 정렬
키워드: Css text center/align property
Text-align: center;

